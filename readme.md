# Talaria Recentlyviewed Service

Some description about application

## Stack 
- Golang
- MongoDB

## Installation

Initiate application only first time

```
./cli init
```

Start service docker container

```
./cli start
```

## Run command inside container

```
./cli {command}
```

## Generate docs
```
./cli docs
```

## Run test 
```
./cli test 
```
